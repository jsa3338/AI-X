[깃허브 사용 방법]
1. git-scm 설치 (cmd에서 git 명령어 사용가능)
2. git bash에서 환경설정
 > git config --global user.name "내아이디"
 > git config --global user.email "내메일주소"
3. git 파일 올리기
 > git add.
 > git commit -m "버전명"
 > git push -u origin main
4. git hub 동기화
 > (최초 집에 세팅) git clone https://github.com/jsa3338/AI-X.git
 > (git>PC 싱크) git pull origin main
