4장. 제어문

1절. 조건문(if)
	if 조건1:
	    조건1이 참일 경우 수행할 명령어(들)
	elif 조건2:
	    조건1이 거짓이고, 조건2가 참일 경우 수행할 명령어(들)
	else:
	    모든 조건을 만족하지 않을때 수행할 명령어(들)
	# elif, else 생략가능

2절. 반복문
   2.1 for - in 문
	for 임시변수 in 나열가능한자료:
	    반복할명령어1
	    반복할명령어2
	    if 조건:
	        break
	else:
	    나열가능한자료가 다 소모되어 for블록을 다 수행한 후 **최종 한 번 수행할 명령어**
	    for문 안에 break를 수행할 경우 else블록은 수행되지 않음
  2.2 while 문
	while 조건:
	    조건이 참일 경우 반복할 문장들
	else:
	    조건이 거짓일 경우 한번 실행할 문장(else구문은 선택사항)

  2.3 break, continue
	break : 이전 단계의 반복문을 탈출
	continue : while 블록 수행을 중단하고 조건으로 감, 즉 건너뛰

3절. 중첩루프
	- 반복문 안의 반복문
	- 중첩루프를 탈출 할 때는 flag 변수나 예외발생을 이용
