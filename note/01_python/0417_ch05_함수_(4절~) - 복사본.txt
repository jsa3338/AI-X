		명령어들
		return 반환값
	변수명 = 함수명  # 함수명 변경

3절. 함수의 매개변수
	- 매개변수에 기본값을 가질 때 : 호출시 선택적으로 명시
		- 매개변수의 기본값이 스칼라 변수일때 : 기본값은 함수 정의 시점으로 세팅
		- 매개변수의 기본값이 객체일때 : 기본값은 함수 호출 시점으로 세팅
	- 키워드 인수 : 기본값을 갖고 호출시 매개변수 명을 반드시 기입해야하는 경우
		- ex) print(.... , sep=' ')
	- 튜플매개변수 : 가변인수를 구현 *args(함수 정의)
	- 딕셔너리매개변수 : 가변적 키워드 인수를 구현 **kwargs(함수 정의)
	- 튜플인수 언패킹 : 함수 호출시 변수 앞에 * 붙임
	- 딕셔너리인수 언패킹 : 함수 호출시 변수 앞에 ** 붙임

    * 함수 정의시 매개변수 순서 *
	순서인수 > 튜플인수 > 키워드인수 > 딕셔너리인수

4절. 람다식
	func_name = lambda 매개변수들 : 리턴값
	func_name(매개변수들)
	(lambda 매개변수들 : 리턴값)(매개변수들)
      * 용도 : (1) 함수의 매개변수   (2) 함수의 return값
	ex1. map(func, 가공할vector)
	ex2. filter(func, 추출할vector)
	ex3. sort(key = 정렬될 데이터를 리턴할 함수, reverse=T/F)
